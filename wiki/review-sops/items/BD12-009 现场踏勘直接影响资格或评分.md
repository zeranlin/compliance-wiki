# BD12-009 现场踏勘直接影响资格或评分

## 标准检查点
现场踏勘直接影响资格或评分

## 检查点定义
审查采购文件是否把参加现场踏勘、现场签到、取得踏勘确认函、提交踏勘证明等事项设置为资格、评分、投标有效性或响应必要条件。

## 权威依据
- 政府采购应保障供应商平等参与，不得设置不合理条件限制、排斥供应商。
- 采购文件可以组织必要现场踏勘，但不得将是否参加踏勘或是否取得踏勘确认材料直接作为资格、投标有效性或评分条件。
- 必要现场信息应通过采购文件、澄清答疑、图纸、现场资料等方式公平公开。

## 适用边界
适用于踏勘、签到、确认函、踏勘证明、踏勘回执等投标前置事项进入资格审查、符合性审查、评分或投标有效性判断的情形。

## 审查执行说明书（3+1）

### 二、组合触发规则

#### 强触发
- 踏勘确认函 / 踏勘证明 / 现场签到 + 必须提供 / 必须参加 + 否定投标人资格。
- 现场踏勘 / 踏勘现场 + 未参与 / 未参加 / 未提供 + 投标无效 / 资格审查不通过 / 不得分。
- 现场踏勘确认函 + 未参与踏勘 + 否定投标人资格。
- 踏勘签到 / 踏勘证明 / 踏勘确认函 + 得分 / 加分 / 不得分。

#### 中触发
- 统一组织现场踏勘 + 应当参加，但未说明未参加后果。
- 要求提交踏勘相关材料，但不清楚是否影响资格、评分或投标有效性。
- 一处写自愿踏勘，另一处写必须踏勘，但后果不明确。

#### 弱触发
- 仅出现现场踏勘、踏勘现场或踏勘费用自理。
- 仅说明踏勘中口头承诺无效。
- 仅说明采购人可以组织踏勘。

#### 不触发
- 明确写明未参与踏勘不作为否定投标人资格的理由，且没有其他相反条款。
- 仅为通用投标人须知模板说明，未要求参加、签到或提交证明。

### 三、审查推理规则

#### 命中规则
- 必须参加现场踏勘或必须提交踏勘确认函，未参加或未提交会导致否定投标资格、投标无效、资格审查不通过、符合性审查不通过或不得分。
- 专用条款要求踏勘影响资格，但通用条款写明不参加不影响资格；若专用条款后果明确为否定资格或投标无效，认定命中。
- 踏勘签到、踏勘证明或踏勘确认函直接进入评分，未提供不得分，认定命中。

#### 待人工复核规则
- 文件要求统一踏勘或应当参加，但没有写明未参加后果。
- 文件要求提交踏勘资料，但不清楚是否影响资格、评分或投标有效性。
- 文件存在自愿踏勘和必须踏勘两类表述，但无法判断哪个条款优先。

#### 不命中规则
- 仅说明采购人可以组织现场踏勘，供应商自愿参加。
- 明确写明未参与踏勘现场不作为否定投标人资格的理由，且没有其他相反条款。
- 仅说明踏勘费用自理、现场资料以书面澄清为准、口头承诺无效。

### 四、质量控制与验证规则

#### 必须提取的证据
- 踏勘要求原文。
- 是否必须参加、签到或提交确认材料的原文。
- 未参加、未签到或未提供材料的后果原文。
- 是否存在相反条款，例如不参加不影响资格。

#### 必须检查的排除条件
- 是否只是自愿踏勘。
- 是否明确不影响资格、评分或投标有效性。
- 是否只是口头承诺无效、踏勘费用自理等程序性说明。
- 是否存在专用条款与通用条款冲突。

#### 常见漏判
- 只看到踏勘对象词，没有识别未参与后的资格后果。
- 表格短句中的踏勘确认函和否定资格未被召回。
- 被后文通用模板条款错误冲淡前文专用风险条款。

#### 常见误判
- 看到现场踏勘就命中。
- 看到未参与踏勘不作为否定资格理由仍然命中。
- 把现场资料以书面澄清为准误判为限制供应商。

#### 最小验证样例
- 正例：投标人必须参与现场踏勘并提供现场踏勘确认函，未参与踏勘现场的将否定投标人资格。
- 反例：采购人可组织现场踏勘，投标人自愿参加，未参与踏勘现场不作为否定投标人资格的理由。
- 边界例：采购人统一组织现场踏勘，投标人应当参加，但未说明不参加后果。

#### 执行步骤
1. 先检索现场踏勘、踏勘确认函、踏勘证明、现场签到、踏勘回执等定位词。
2. 提取候选条款，并读取资格审查、符合性审查、评分办法、投标文件组成、响应格式和投标人须知上下文。
3. 判断是否要求必须参加、签到或提交踏勘确认材料。
4. 判断未参加、未签到或未提交材料是否影响资格、评分、投标有效性或材料认可。
5. 检查是否存在“不参加不影响资格、评分或投标有效性”的排除条款。
6. 如专用条款与通用条款冲突，优先看专用条款是否明确设置否定资格、投标无效或不得分后果。
7. 按命中、待人工复核、不命中输出结论。
