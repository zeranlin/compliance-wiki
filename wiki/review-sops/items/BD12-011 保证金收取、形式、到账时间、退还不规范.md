# BD12-011 保证金收取、形式、到账时间、退还不规范

## 标准检查点
保证金收取、形式、到账时间、退还不规范

## 检查点定义
审查投标保证金、履约保证金的金额比例、提交形式、到账时间、退还方式、退还时间、退还条件和不予退还情形是否符合法定要求。

## 权威依据
- 《中华人民共和国政府采购法实施条例》第四十八条：采购文件要求中标或者成交供应商提交履约保证金的，履约保证金数额不得超过政府采购合同金额的 10%。
- 财库〔2019〕38号要求规范保证金收取和退还，不得收取没有法律法规依据的保证金。

## 适用边界
适用于投标保证金、履约保证金及其提交形式、金额比例、到账、退还和不予退还规则。质量保证金、预付款担保、维修保证金等不得与履约保证金混同。

### 二、组合触发规则

#### 强触发
- 履约保证金 + 合同金额 / 合同价 / 中标金额 / 成交金额 + 12% / 15% / 百分之十二 / 超过 10%。
- 履约保证金 + 必须提交 / 应提交 / 须提交 + 合同金额的 X%，且 X 大于 10。
- 投标保证金 + 超过采购项目预算金额 2%。
- 保证金 + 现金 / 转账唯一形式 + 不允许保函等非现金替代。

#### 中触发
- 要求提交履约保证金，但未明确金额、比例、形式、退还时间或退还条件。
- 要求保证金到账，但到账时间与投标截止时间、签约时间关系不清。
- 合同草案和投标人须知对保证金比例或退还规则不一致。

#### 弱触发
- 仅出现保证金提示，但没有金额、比例、形式、到账或退还约束。
- 仅提示依法执行保证金政策。

#### 不触发
- 履约保证金比例不超过合同金额 10%，且允许非现金形式提交。
- 空白保证金条款仅出现在明确仅供参考的合同模板或示范文本中，且正式条款已补足。

### 三、审查推理规则

#### 命中规则
- 正式采购文件、商务要求、投标人须知、专用合同条款或对乙方有约束力的合同草案要求履约保证金超过政府采购合同金额 10%。
- 条款明确写成履约保证金为合同金额的 12%、中标金额的 15% 或成交金额百分之十二等超过 10% 的比例。
- 要求以现金、转账等单一方式提交保证金，未允许支票、汇票、本票、保函等非现金形式。
- 收取没有法律法规依据的保证金。

#### 待人工复核规则
- 仅写应提交履约保证金，但比例、金额、退还方式或退还时间不清。
- 条款位于合同模板、格式文本或示范文本中，无法确认是否作为正式专用条款约束供应商。
- 投标人须知、商务要求和合同草案对保证金口径不一致，但优先级不清。

#### 不命中规则
- 履约保证金比例等于或低于合同金额 10%，并允许非现金形式提交。
- 仅为依法执行保证金政策的提示性表述。
- 空白条款仅位于明确仅供参考的合同模板中，且正式条款已有合法口径。

### 四、质量控制与验证规则

#### 必须提取的证据
- 保证金类型原文，区分投标保证金和履约保证金。
- 金额、比例和计算基准原文。
- 提交形式原文。
- 退还方式、退还时间、退还条件和不予退还情形原文。
- 所在章节是否为正式条款或仅供参考模板。

#### 必须检查的排除条件
- 是否把投标保证金、履约保证金、质量保证金或维修保证金混同。
- 是否超过履约保证金 10% 上限。
- 是否只是模板空白。
- 是否允许非现金形式提交。
- 是否有其他正式章节补足合法口径。

#### 常见漏判
- 看到履约保证金比例清楚，就忽略超过 10% 的硬性上限。
- 被后文合同模板空白冲淡前文正式条款中的超比例要求。
- 未识别合同金额、中标金额、成交金额都是比例基准。

#### 常见误判
- 把仅供参考合同模板中的空白保证金直接判为命中。
- 把比例低于或等于 10% 的履约保证金直接判为违规。
- 把质量保证金、维修保证金误判为履约保证金。

#### 最小验证样例
- 正例：乙方应在签订本合同之日提交履约保证金为合同金额的 12%。
- 反例：履约保证金不超过合同金额的 10%，并允许保函等非现金形式提交。
- 边界例：合同格式写履约保证金____元，退还期限____日，但该格式标注仅供参考。

#### 执行步骤
1. 先检索投标保证金、履约保证金、质量保证金、维修保证金、担保、保函、退还等定位词。
2. 提取候选条款，并读取投标人须知、商务要求、合同草案、专用合同条款和保证金章节上下文。
3. 区分保证金类型，避免把投标保证金、履约保证金、质量保证金或维修保证金混同。
4. 提取金额、比例、计算基准、提交形式、到账时间、退还时间和不予退还条件。
5. 对履约保证金重点检查是否超过政府采购合同金额 10%。
6. 检查是否允许支票、汇票、本票、保函等非现金形式，是否只是模板空白或示范文本。
7. 按命中、待人工复核、不命中输出结论。
