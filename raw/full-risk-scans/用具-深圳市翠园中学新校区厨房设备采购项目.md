# 用具-深圳市翠园中学新校区厨房设备采购项目 主审查页

## 主审查结论
- 本项目属于货物类公开招标，厨房设备参数、评分项和证明资料链条较重，已识别出 7 类应优先关注的政府采购合规风险。
- 本项目最突出的风险不在单一条款，而在“技术参数高密度证明材料要求 + 评分项中叠加认证/责任险/业绩/方案主观分档 + 平台截图与原件备查绑定”的组合效应，容易把供应商既有资质储备和形式举证能力放大为竞争优势。
- 现阶段可优先关注：格式性事项转技术扣分、方案分主观分档、产品责任险直接评分、认证评分堆叠、检测报告与认e云截图过重、原件备查与无法判断不得分、专门面向中小企业项目的政策口径未闭环。

## 风险点 1：将“规格/品牌/型号”填写错误直接折算为技术负偏离
- 风险等级：高
- 项目具体风险：
  招标文件把《分项报价表》中的“规格/品牌/型号”填写错误或缺漏，直接按“一项一般技术参数负偏离”扣分处理，实质上把格式性或填报性错误转化为技术评分后果。
- 风险原因：
  “规格/品牌/型号”栏位是否填写完整，通常首先属于投标文件编制和响应准确性问题；若未导致采购需求无法识别，一般不宜直接等同于技术性能不满足。该写法会把形式问题变成实质技术扣分。
- 行号证据：
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:129`：技术要求符合性评价按技术参数负偏离扣分。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:130`：明确写明“规格/品牌/型号”填写错误或缺漏按一项一般技术参数负偏离扣分。
- finding 映射：
  - [[findings/格式性事项不得作为评分条件|格式性事项不得作为评分条件]]
- 法规/规则映射：
  - [[legal-bridges/采购需求|采购需求]]
- 边界说明：
  若报价表关键内容缺失到足以影响产品识别、报价比较或实质响应判断，采购人可以按响应完整性处理；但本条当前写法是直接折算为技术负偏离，风险主要在于处理方式过重。

## 风险点 2：设计、安装方案评分采用主观分档，缺少可复核量化边界
- 风险等级：高
- 项目具体风险：
  设计、安装方案评价先按“满足几项内容”给基础分，再由评审委员会按“极合理、合理、较合理、一般合理、不合理”等主观表述进行二次分档，量化边界不足。
- 风险原因：
  条款虽然列出平面图、安装条件图、效果图、安装方案四类内容，但高低档之间仍主要依赖“条理清晰”“可操作性强”等主观表达，缺少明确、可复核的客观标准，不利于不同评委按同一尺度执行。
- 行号证据：
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:133`：评分项为“设计、安装方案评价”。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:141`：按满足四点、三点、两点、一点分层给基础分。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:142`-`148`：进一步按“极合理、合理、较合理、一般合理”等主观档位评分。
- finding 映射：
  - [[findings/评分项未细化量化|评分项未细化量化]]
- 法规/规则映射：
  - [[legal-bridges/采购需求|采购需求]]
- 边界说明：
  方案类内容并非当然不能评分；如果每档均写清楚图纸完整度、安装条件识别深度、重难点覆盖率、风险应对项数等客观边界，风险可明显下降。

## 风险点 3：高额产品责任险被直接计分，并叠加特定查询截图要求
- 风险等级：中高
- 项目具体风险：
  招标文件把两类燃气灶具的第三方产品责任险作为 8% 权重评分项，且要求累计赔偿限额、单次事故赔偿限额等达到较高数值，同时要求提供保单扫描件和 APP 查询截图。
- 风险原因：
  产品责任险可以作为风险管理辅助信息，但在货物采购评分中直接赋予较高分值，容易放大既有保险资源储备优势；保险限额高低也并不当然等于本项目交付质量、安装质量或售后履约能力。
- 行号证据：
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:151`-`157`：产品责任险评价占分，并对两类产品设置较高赔偿限额门槛。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:159`：要求提供保单扫描件及保单出具方 APP 查询截图。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:160`：未按要求提供或无法判断的不计得分。
- finding 映射：
  - [[findings/证书设置与项目相关性不足|证书设置与项目相关性不足]]
  - [[findings/查询截图或平台核验要求过细|查询截图或平台核验要求过细]]
- 法规/规则映射：
  - [[legal-bridges/采购需求|采购需求]]
- 边界说明：
  若采购人确有高风险场景考虑，可在履约或合同责任安排中要求中标人投保；但在投标评分阶段以高额责任险直接拉开分差，需要更充分论证其与本项目履约质量的直接关系。

## 风险点 4：供应商认证情况评分堆叠，容易放大既有认证储备优势
- 风险等级：中高
- 项目具体风险：
  文件把多个具体产品的有毒有害物质限量认证、防变形性能认证、产品安全认证、防腐蚀等级认证、防水等级认证、阻燃等级认证等直接纳入 10% 权重评分。
- 风险原因：
  这类认证在部分设备上可能具有一定参考价值，但当前条款采用“多证书累计给分”的方式，且证书名称需与所投产品名称相符，容易把既有认证储备和特定产品路线优势转化为高分，而不是围绕本项目核心履约结果进行评价。
- 行号证据：
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:208`-`215`：供应商认证情况占 10%，多类产品认证分项累计给分。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:217`：要求证书扫描件与官网或全国认证认可信息公共服务平台查询截图同时提供，缺一不得分。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:218`：无法判断是否符合评分要求的一律不得分。
- finding 映射：
  - [[findings/证书设置与项目相关性不足|证书设置与项目相关性不足]]
  - [[findings/查询截图或平台核验要求过细|查询截图或平台核验要求过细]]
- 法规/规则映射：
  - [[legal-bridges/采购需求|采购需求]]
- 边界说明：
  若个别认证确与核心安全风险直接对应，可以考虑降权并允许等效证明；但像当前这样多项叠加计分，更接近“资质储备竞争”，而不是“采购需求导向评价”。

## 风险点 5：大量重要技术参数和不可负偏离条款绑定 CMA 报告、认e云截图与特定检测路径
- 风险等级：高
- 项目具体风险：
  用户需求书中大量 `★` 和 `▲` 技术条款要求投标时同步提交带 `CMA` 标志的检测报告、认证证书、认e云查询截图，并要求在报告中逐项标注检测项、样品照片；部分条款还将不满足直接对应为投标无效或重点扣分。
- 风险原因：
  这种写法不是针对少数关键安全底线，而是高频扩散到多个设备、多个核心部件和多个检测项目，证明链明显过重。供应商能否在投标阶段持有完全对应的特定型号检测资料，容易取决于既有样机、认证储备和预制资料能力，而不完全是履约能力本身。
- 行号证据：
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:2333`-`2334`：紫外线消毒器条款要求 `CMA` 报告和认e云截图，不满足可导致投标无效。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:2476`、`2479`、`2512`、`2530`、`2533`、`2536`：多个重要参数要求 `CMA` 报告、认证证书及认e云截图。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:2806`、`3208`、`3331`、`3334`、`3781`、`3784`、`3829`：不同设备继续重复相同证明路径。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:11521`-`11546`：商用洗碗机等条款继续叠加多组检测报告、认证证书和认e云截图要求。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:13605`-`13606`：未按要求提供证明资料、证明资料模糊不清或编号位置不一致，可被判为负偏离。
- finding 映射：
  - [[findings/检测报告要求与评审必要性不匹配|检测报告要求与评审必要性不匹配]]
  - [[findings/查询截图或平台核验要求过细|查询截图或平台核验要求过细]]
  - [[findings/原件备查与评审可操作性不足|原件备查与评审可操作性不足]]
- 法规/规则映射：
  - [[legal-bridges/采购需求|采购需求]]
- 边界说明：
  对少数直接关系食品安全、燃气安全、电气安全的核心指标，要求检测证明并非当然不可以；本项目的主要问题在于覆盖面过广、证明路径过窄，且大量绑定到投标阶段即时提供。

## 风险点 6：业绩和证明材料普遍采用“原件备查 + 无法判断不得分/判负偏离”，评审过度依赖形式完备度
- 风险等级：中高
- 项目具体风险：
  无论是责任险评分、同类业绩评分、认证评分，还是技术参数证明资料规则，均普遍采用“原件备查”“专家无法判断不得分”“证明资料模糊不清判负偏离”的处理方式。
- 风险原因：
  政府采购评审当然需要核验证明材料，但若在多个评分项中普遍以“无法判断”为由直接不得分，且未提供更稳定的核验路径，就容易把形式材料清晰度和编排能力过度放大为得分差异。
- 行号证据：
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:159`-`160`：责任险评分要求原件备查，无法判断不计得分。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:205`-`206`：业绩要求合同、清单、验收报告、履约评价齐备，无法判断一律不得分。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:217`-`218`：认证评分要求截图与证书同时提供，无法判断不得分。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:14421`-`14455`：通用条款明确，评分项材料不清晰、异常或无效的，可直接按 0 分处理。
- finding 映射：
  - [[findings/原件备查与评审可操作性不足|原件备查与评审可操作性不足]]
- 法规/规则映射：
  - [[legal-bridges/采购需求|采购需求]]
- 边界说明：
  `14421` 对“原件备查”的解释表明，原件核查本意主要是质疑、投诉等阶段的后续核查；但在评分条款中被广泛前置为即时扣分后果，会放大评审不稳定性。

## 风险点 7：专门面向中小企业采购项目仍保留“非专门面向项目价格扣除 / %”模板口径，政策表达未闭环
- 风险等级：中
- 项目具体风险：
  文件前文明确写明本项目“专门面向中小企业采购”，但后续“关于享受优惠政策的主体、价格扣除比例及采购标的所属行业”部分仍保留“非专门面向中小企业采购的项目，应执行价格扣除比例……给予 / % 的扣除”的模板条款，未完全清理。
- 风险原因：
  该问题未必直接影响本项目资格审查，但会造成文件逻辑不够闭环，影响投标人对价格政策适用范围的理解，也反映出模板切换不完整。
- 行号证据：
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:303`：明确写明“本项目是（否）专门面向中小企业采购：是”。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:375`：专门面向中小企业采购的项目不再执行价格扣除比例。
  - `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:376`：仍保留“非专门面向中小企业采购的项目，应执行价格扣除比例……给予 / % 的扣除”。
- finding 映射：
  - 暂不升级为稳定 finding，先保留为本项目模板闭环风险。
- 法规/规则映射：
  - [[legal-bridges/采购需求|采购需求]]
- 边界说明：
  若采购公告、资格条款、评标价格公式最终均按“专门面向中小企业”执行，实质后果可能可控；但作为正式招标文件，政策表述仍应闭环，避免留下理解分歧。

## 本次主审查未入库事项
- `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:203` 的“同类业绩（厨房设备采购）”写法，当前更像货物品类贴合度要求，是否上升为“特定行业业绩作为加分条件”，需结合采购清单、设备构成和同类项目外延再复核。
- `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:304` 的“燃气器具安装维修”资质要求，因本项目包含燃气设备安装内容，是否属于合理履约资质，需结合安装范围和当地燃气管理口径进一步判断，本次未直接入库为风险点。
- `raw/numbered-text/用具-深圳市翠园中学新校区厨房设备采购项目.md:233`-`234` 的诚信查询条款由评审工作人员统一查询，当前未见把平台截图责任转嫁给投标人，本次不作为风险点入库。
