## 3. 计算口径

- 忽略新增数：标准答案名称中带“（新增）”或“(新增)”且不属于当前 151 个 NBD 覆盖范围的项，不进入 F1 分母。
- 工程输出数：只统计模型输出中进入评测的“命中”结果，不统计“待人工复核”。
- 召回率 = 匹配成功数 / 标准答案数
- 精确率 = 匹配成功数 / 工程输出数
- F1 = 2 * 召回率 * 精确率 / (召回率 + 精确率)

## 4. 结论

这条线的核心演进是：

1. 先把原始标准答案整理成可评测的 V2。
2. 再把工程真风险补入 V3，提升覆盖率。
3. 最后把 V3 中证据不足或错位的内容裁掉，形成 V4。
